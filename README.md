# 喝水记录

轻量 Android 原生 Kotlin App，用于本地记录每日饮水量、提醒喝水，并提供 1x4 透明桌面小组件。

## 技术栈

- Kotlin
- 原生 Android View
- SharedPreferences + JSON 本地记录
- AlarmManager 提醒
- RemoteViews 桌面小组件

不使用 Compose、Room、DataStore、WorkManager、Glance，目标是保持 APK 体积小。

## 功能

- 首页显示今日饮水量和进度。
- 快捷记录：+100ml、+200ml、+300ml、+500ml、自定义。
- 首页 4 个快捷记录量可以在设置里自定义。
- 支持删除单条今日记录。
- 支持撤销上一次记录。
- 设置每日目标，默认 2000ml。
- 支持按星期设置每日目标，例如工作日和周末使用不同目标。
- 设置默认饮水量，默认 500ml，小组件按钮会同步使用。
- 设置提醒间隔：15、30、45、60、90、120 分钟。
- 设置提醒开始和结束时间，默认 08:00 到 23:00。
- Android 13 及以上申请通知权限。
- 达到每日目标后，自动停止当天提醒。
- 首页显示每天首次达到目标的时间。
- 首页显示当前喝水进度是否偏慢，并按提醒时间段估算建议补充量。
- 超过每日目标后继续记录，并显示已超出多少 ml。
- 如果 30 分钟内刚记录过饮水，本次提醒会自动跳过，避免刚喝完又提醒。
- 每天零点自动刷新小组件，切换到新一天数据。
- 统计最近 7 天和 30 天：平均饮水量、达标天数、达标率、最高一天、连续达标天数。
- 统计今天上午、下午、晚上的喝水量分布。
- 首页会提示连续多天未达标。
- 设置页显示已记录天数和记录条数。
- 超过 180 天的详细记录会自动整理为每日总量，长期统计仍保留。
- 1x4 透明小组件：白色文字、进度条、右侧快捷按钮。
- 小组件支持大字体开关和水杯图标显示开关。

## 本地存储

App 不需要登录，不联网。饮水记录和设置都保存在本机 SharedPreferences 中。
