# 喝水提醒

这是一个本地运行的 Android 喝水提醒 App。

## 已实现功能

- 设置每日喝水目标，默认 2000 ml
- 设置提醒间隔，默认 60 分钟
- 设置提醒时间段，默认 09:00 到 22:00
- 通知提醒喝水
- 快速记录 100 / 200 / 250 / 500 ml
- 自定义记录每次喝水量
- 撤销今天最后一次记录
- 统计今日进度、连续达标、7 天平均、30 天平均、30 天达标天数、累计记录天数和累计喝水量
- 查看今天记录和最近 30 天记录
- 透明桌面小组件，显示今日饮水量，并支持快捷 +250ml

## 说明

数据只保存在手机本地，不上传服务器。

Android 13 及以上需要允许通知权限。部分国产手机还需要在系统设置里允许后台运行或自启动，否则系统可能延迟提醒。

当前项目未使用你发来的 GitHub token，也不会推送到 GitHub。那个 token 已经暴露，建议在 GitHub 里撤销后重新生成。
